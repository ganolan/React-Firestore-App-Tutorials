# React + Firebase Security & Authentication Tutorial Series

This series builds on the Todo App and Bulletin Board tutorials. Students learn how to protect their Firebase project and control who can access their data.

## What You Will Learn

- Why your Firebase API key is safe to commit — and what the real risks actually are
- How Firebase Security Rules work and how to write them
- How to add Google sign-in to a React app so only authenticated users can read and write data

## Why This Matters

Every app built so far has had an open database — anyone with the config can read or write data. Before sharing a project publicly, students need to understand:

- What Firebase Security Rules are and how to set them
- What authentication is and how to wire it up in React
