# React + Firebase Bulletin Board Tutorial Series

This series builds on the Todo App tutorials. Students now design their own Firestore data model and connect a more complex, multi-screen app to a cloud database.

## What You Will Build

A **Class Bulletin Board** where students can:
- Post messages and project updates
- Click into a post to read it in full
- Leave comments on any post

## Why This Project?

The Todo app used a single Firestore collection with identical documents. Real projects need **multiple connected collections**. This series teaches students how to:

- Read a working app and figure out what data it needs
- Design a Firestore structure before writing any code
- Query across collections using a shared ID

## The Approach

You are given a **working starter app** with all the UI already built and realistic fake data. Your job across the three tutorials is to replace the fake data with real Firestore connections.
